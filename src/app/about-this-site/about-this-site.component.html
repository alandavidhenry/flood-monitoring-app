About this site

This website was created as a submission for the jHub Coding Scheme which aims "to give serving personnel the opportunity to gain valuable coding and AI skills, whilst getting paid to do so, so that they might use these skills to enable an information advantage".

This particular challenge is to create a dynamic website that makes multiple calls to a Government API. It must also use a framework. I chose to use the Angular Framework after completing Maximilian Schwarzm√ºller's Angular - The Complete Guide course on Udemy.

This website was created using:

  • Angular
  • Tailwind
  • Flowbite

I have used the Government Flood Warning API to fetch data about current flood warnings and to list all the areas included in the API.

The flood warning data is fetched and displayed in a form that can be sorted and filtered.

The areas can be displayed within a radius of a postcode. To convert the post codes to latitude and longitude, I used the Postcode & Geolocation API for the UK API.
